# Etsy Listing Templates & POD Automation

Last updated: February 14, 2026

## Title Template

Format: `[Design Theme] [Product Type]: [Secondary Keywords], [Tertiary Keywords]`
Max: 140 characters. Use ALL of them.
Rule: Primary keyword in first 3 words. No filler words.

Example: `Celestial Sweatshirt: Boho Moon Phase, Mystical Sun Hoodie, Boho Celestial Gift`

## Pricing Formula

```
sale_price = blank_cost / (1 - target_margin - etsy_fee_rate)
listed_price = sale_price * 2  (for perpetual 50% sale)
etsy_fee_rate = 0.13 (6.5% transaction + 3% + $0.25 payment + $0.20 listing)
```

## Multi-Product Listing Strategy

Each listing should offer 3 product types as variants:
- T-Shirt (cheapest entry point)
- Crewneck Sweatshirt (mid-range)
- Hoodie (highest price)

Each with sizes S through 2XL = 15 SKUs per listing minimum.
Add 10-15 color options = massive search surface area.
