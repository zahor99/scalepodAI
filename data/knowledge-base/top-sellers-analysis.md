## Key Success Patterns

1. **Perpetual 50% sale** - List at 2x, run permanent "sale" (ISHIRTLAB model)
2. **Multi-product listings** - T-shirt + sweatshirt + hoodie as variants in one listing
3. **15+ color variants** - More colors = more search surface area
4. **Star Seller badge** - Fast dispatch, good reviews, quick replies
5. **Niche consistency** - Pick ONE aesthetic and own it (TheCocoBear model)
6. **High listing count** - CozyThreadsCustoms has 1,670 listings
7. **Seasonal launches** - New designs for every holiday/season
8. **Backprint designs** - Growing trend, commands premium (KinderPlanetCompany)
