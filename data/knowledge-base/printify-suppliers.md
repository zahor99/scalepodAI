## Etsy Fee Stack

- Transaction fee: 6.5%
- Payment processing: 3% + $0.25
- Listing fee: $0.20 per listing
- Offsite ads: 15% (if applicable, opt-out possible under $10K revenue)
- **Total effective fees: ~9.75-10% of sale price** (without offsite ads)
- Note: Previous estimate of ~13% was overstated. Actual fee on $74.99 sale = $7.57 (10.1%)

## API Constraints

- Printify: 200 requests per 30 minutes, max 50MB images
- Etsy: OAuth 2.0, tags max 20 chars each, exactly 13 tags
- Image Gen: Must reach 3000x4000px @ 300 DPI minimum for print
